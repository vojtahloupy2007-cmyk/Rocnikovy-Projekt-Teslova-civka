# Ročnikový-Projekt-Teslova-cívka

#  Zadání projektu: Teslova cívka – Ověření pravidla pravé ruky a bezdrátový přenos energie

##  Cíl projektu
Cílem projektu je:
- Ověřit **pravidlo pravé ruky pro cívku** pomocí pozorování orientace magnetického pole.
- Demonstrovat **přenos energie vzduchem** prostřednictvím vysokofrekvenčního elektromagnetického pole.
- Postavit Teslovu cívku


# Co je pravidlo pravé ruky pro cívku
-Pravidlo pravé ruky pro cívku slouží k určení směru magnetického pole a polarity cívky
- U tesloví cívky je ze uchop cívku pravou rukou
Prsty ohni ve směru proudu v závitech
Palec ukazuje směr magnetického pole uvnitř cívky
Směr palce zároveň určuje severní pól cívky
