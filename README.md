# Rocnikovy-Projekt-Teslova-civka

#  Zadání projektu: Teslaova cívka – Ověření pravidla pravé ruky a bezdrátový přenos energie

##  Cíl projektu
Cílem projektu je:
- Ověřit **pravidlo pravé ruky pro cívku** pomocí pozorování orientace magnetického pole.
- Demonstrovat **kvalitativní přenos energie vzduchem** prostřednictvím vysokofrekvenčního elektromagnetického pole.
- Pochopit fyzikální principy, které stojí za elektromagnetickou indukcí a rezonancí.
